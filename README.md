## Pre-requisites

## Debian

```bash
# minimal server libvirt install
sudo apt install --no-install-recommends qemu-system libvirt-clients libvirt-daemon-system

# utils
sudo apt install cloud-image-utils qemu-utils
```

## Install

```
curl -fsSL https://raw.githubusercontent.com/natebuckareff/vmlot/main/install.sh | sudo bash
```
